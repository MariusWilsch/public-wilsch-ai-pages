# Case Study Slide Template
[[sales-pipeline]]

A reusable 6-slide template for B2B AI/technology case studies. Based on consulting-standard structures (McKinsey/BCG/Bain) adapted for mixed audiences (IT managers + C-level executives).

---

## Usage Notes

- **Audience strategy:** Business value first (Slides 1-4), technical details second (Slide 5)
- **Design principle:** One big idea per slide, headline tells story without reading body
- **Output formats:** Presentable slides + PDF for sending
- **Pyramid principle:** Start with conclusion, then supporting evidence
